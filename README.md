## Description

[Template-Nest-API](https://github.com/cledsonfs-cmd/Template-Nest-API): Template para o desenvolvimento de APIs com base no framework Nest.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
$ docker compose up -d
```
